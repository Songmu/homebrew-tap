# Songmu/tap

Third party homebrew tap to install the tools Songmu is involved in developing.

## Synopsys

```console
# tap and install
% brew tap Songmu/tap
% brew install gocredits

# install directly
% brew install Songmu/tap/gocredits
```

## Tools we provide
- blogsync
- ecschedule
- gh2changelog
- ghg
- gocredits
- godzil
- goxz
- maltmill
- peep
